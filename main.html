Matterport + MaintainX Integration Guide

This guide outlines how to integrate MaintainX work orders and assets with Matterport’s Tags, Notes, and Search capabilities to streamline field operations and facility maintenance.

Step 1: Scan the Facility

Use a Matterport Pro3 or compatible camera to scan the entire facility. Ensure complete coverage of critical areas such as mechanical rooms, utility spaces, and equipment zones.

  • Upload the model to Matterport Cloud.
  • Apply floor and room labels to improve navigation.

Step 2: Set Up Assets & Work Orders in MaintainX

Create digital records in MaintainX for each asset (e.g., HVAC units, panels, pumps).

  • Add asset details: name, serial number, manuals, service history.
  • Create work orders tied to those assets.
  • Copy shareable links for each asset or work order (ensure appropriate access permissions).

Step 3: Place Tags with Work Order Details

Inside Matterport Workshop:

  • Navigate to each asset’s physical location.
  • Add a Tag.
  • Use a consistent naming format in the title:
AHU-3 | Filter PM | Due 2025-06-01

In the tag description, include:

  • A summary of the task
  • Hyperlinked MaintainX Work Order:
[Open Work Order](https://app.getmaintainx.com/workorders/MX123456)
  • Key checklist items or safety notes

Step 4: Use Matterport’s ‘Search Within Model’ Feature

Technicians can use the Matterport search bar to locate:

  • Asset IDs (e.g., AHU-3)
  • Task types (e.g., Vibration Inspection)
  • Work Order numbers (e.g., MX123456)
  • Due dates or priority labels if included in tags

Best Practice: Repeat keywords in both the Title and Description of each Tag to optimize searchability.

Step 5: Use Notes for Collaboration

Use Matterport Notes for asynchronous communication or quality control:

  • Add Notes for inspections, open issues, or punchlist items.
  • Tag team members or assign follow-ups if supported.
  • Use Notes to record issues that arise during walkthroughs or repairs.

Step 6: Share Model for Execution

Distribute the Matterport model link to your field or maintenance team.

  • Train them to:
    • Use Search to quickly locate tasks
    • Click Tags to open related MaintainX work orders
    • Leave Notes for unresolved issues or clarification

Teams can use the 3D model for spatial orientation, reducing task time and miscommunication.
